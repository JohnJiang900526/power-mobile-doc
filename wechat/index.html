# PowerMobile 企业微信配置文档

# 开通企业微信

  1. 目前企业微信与原企业号已经合并，登录企业微信官网让客户自行注册即可。 企业微信官网地址，界面如下：

2.注册成功后，会自动生成企业ID，记录下来，需要在后面PMS应用站点配置，界面如下图：

3.导入通讯录。在通讯录菜单下，可以创建部门，批量导入成员，并邀请用户关注，可通过短信或邮件或微信号批量邀请关注,界面如图：

# 配置企业应用

1.进入企业微信后台，进入企业应用——自建应用，点击新建应用，创建应用，应用名称会出现在微信通讯录，请客户想好名称。

2.创建应用后，会生成应用的AgentId和Secret码，复制出来，后面配置会用到。点击编辑按钮，可以编辑该应用的可见范围，可以选择人员或部门。

3.为应用设置可信域名。点击“网页授权及JS-SDK”，输入PMS站点域名，点击申请校验按钮。注意，此处一定要是域名才行，IP地址是不行的。

校验完成后，会产生一个txt文档，下载后放到PMS站点根目录下。

4.配置菜单，进入应用，点击自定义菜单，为应用配置默认菜单

目前PowerMS提供了一个默认菜单[进入系统]，项目上可以根据自己的喜好进行命名，网址配置为：你的网站网址/PowerMobile2/Reg.ashx

# 配置站点中的企业微信参数

1.双击项目的根目录下面的Power.ConfigEdit.exe文件，选择微信参数 界面如下：

1).CorpId代表的是企业微信的企业ID

2).Secret代表的是自建应用的Secret

3).应用ID代表的是自建应用的AgentId

4).站点外网访问地址必须有http://前缀，不可省略，不然安卓消息推送会有问题。

到此步骤，已经可以实现微信端接收PMS推送的消息。
