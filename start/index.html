# 初识 Power-Mobile

Power-Mobile APP 是原生程序结合h5框架Vue.js形成的混合开发APP。

原生程序主要包含html5plus和上海普华拓展的原生程序。

h5框架使用的是Vue.js，Vue组件库使用的是Vant。

# Power-Mobile APP 和PowerPMS的区别

Power-Mobile APP 和 PowerPMS APP都是混合开发的，也就是说都是原生APP+h5的模式。

PowerPMS的主要tab页面(比如：我的，项目)是无法自定义的，主tab审批是可以编辑的，表单和窗体可以 配置.pho或者htm文件，可以对页面进行自定义。

Power-Mobile的主要界面，从登陆那一刻开始都是可以编辑的，此种编辑是建立在会使用vue的基础上， 如果不会vue的话，自定义是有点难度的。Power-Mobile 也可以自定义配置窗体和表单，就像PowerPMS 中的窗体和表单一样可以配置.pho或者htm文件。

# 总结

Power-Mobile构建的初衷是使用最低廉的技术&人工成本，快速形成一个易拓展、实用性强的APP。当然， 此APP存在存在一种无法避免的缺陷，那就是在无网络环境或者网络环境不好的情况下，APP的体验不好。这 也是所有混合开发APP的缺陷。
