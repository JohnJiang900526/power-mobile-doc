# PowerMobile2 版本更新

注意

下面是更新版本的列表，更新版本的频率是1个月更新发布一次。

# 2020-12-30 beta版

  1. 视频播放组件添加
  2. 电子信息签名组件添加
  3. 优化人员选择模块 可选在当前EPS下获取人员
  4. 修复安卓返回问题
  5. 审批流当时一个时直接默认选中跳转
  6. 动态部分允许自定义页面 嵌入iframe的形式
  7. 登录形式和后端的token保持一致

注意

此版本需要更新M3 12月份的补丁，慎重使用，目前还不稳定，还需要经过大量测试

2020-12-30-beta[版本]补丁包 提取码：msl1

# 2020-12-30 稳定版

  1. 视频播放组件添加
  2. 电子信息签名组件添加
  3. 优化人员选择模块 可选在当前EPS下获取人员
  4. 修复安卓返回问题
  5. 审批流当时一个时直接默认选中跳转
  6. 动态部分允许自定义页面 嵌入iframe的形式

注意

此版本千万不要更新M3 12月份的补丁，后端补丁请使用2020-12月份以前的版本

2020-12-30-lts[版本]补丁包 提取码：t2wk

# 2020-11-30

  1. 企业微信下，调用相机、图库的封装 参考API中的 WeixinUpload
  2. 解决APP上传图片时，个别手机类型上传失败的问题 参考config.js的appUploadMethod属性
  3. 动态模块可以自定义页面，参考config.js的customizeDynamicsUrl属性
  4. 优化流程，当只有一个流程的时候，界面自动选中并跳转
  5. 人员选择模块，添加了从EPS节点下获取人员信息功能
  6. 企业微信下，二维码功能调用 参考API中的 WeixinScanQRCode
  7. 切换站点，添加站点验证
  8. 解决PHO模式的层级比较多的时候，安卓的返回问题
2020-11-30[版本]补丁包 提取码：dfpy

# 2020-10-30

  1. 解决安卓手机在杀死app、锁屏、没有激活app的情况下无法接收到消息推送问题
