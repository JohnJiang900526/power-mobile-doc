# Power-Mobile 版本更新

下面是更新版本的列表，更新版本的频率是1-2个月更新发布一次。

# 2019-12-30[版本]

  1. 在表单中，嵌入iframe， 实现vue程序和iframe页面的方法进行交互[正在测试]。

2019-12-30[版本]补丁包

# 2019-11-30[版本]

  1. 通知模块 [界面优化]
  2. APP消息推送页面跳转优化 [跳转逻辑优化]
  3. 表单权限优化 [主表、子表、附件的编辑和禁止权限优化处理]
  4. 表单多字段input框 [添加双击事件, 弹出类似微信的蒙版]
  5. 审批流添加附件方法 [在审批流中可以上传附件]

2019-11-30[版本]补丁包

# 2019-10-30[版本]

  1. EPS数据的优化 [优化树形数据算法]
  2. 个人中心数据的迁移 [原来是后端控制，目前转义到前端进行控制]
  3. 添加允许头像上传的开关 [可以禁止上传头像]
  4. 消息模块的配置选择 [可以选择聊天模式和非聊天模式]
  5. 工作台菜单检索 [在工作台中可以进行搜索菜单名称]

2019-10-30[版本]补丁包

# 2019-09-30[初始版本]

  1. PowerMobile2的最初发布。
  2. 整个APP的实现
  3. 个人中心功能配置拓展
  4. 消息页面的集合添加
  5. 窗体和表单的实现
  6. 动态模块的开发
  7. 工作台模块的开发
  8. 企业微信的关联

2019-09-30[版本]补丁包
