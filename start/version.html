# PowerMobile2 版本更新

注意

下面是更新版本的列表，更新版本的频率是1个月更新发布一次。

# 2021-05-30

  1. 极光推送 身份别名逻辑优化 解决个别时候消息接收不到的情况
  2. 主题颜色设置 目前涉及到的范围比较简单 只有主题颜色，tabbar, 菜单颜色的设置
  3. 处理pho形式 新增的时候保存数据 页面不能存留的问题
  4. 隐私保护处理
  5. pho页面送审 发出去的数据没有formId bug处理
  6. 页面白屏简单处理
  7. 保存按钮的权限 优化
  8. pho页面实现周期进去反馈 需要更新对应的pho文件(南京公司已经使用)
  9. 优化清除别名的方法
  10. 日期组件优化
  11. 窗体样式修改

注意

此版本后端M3补丁请使用2021-2月份以后的版本, PC移动端的很多功能需要M3最新补丁， 请项目上尽量克服，尽快更新M3的最新补丁。以后不再考虑beta版和稳定版。对老的M3补丁不再继续做兼容。

2021-05-30[版本]补丁包 提取码：rij1

# 2021-03-30

  1. toast组件时长自定义控制
  2. PHO模式下自定义按钮 事件要在PHO页面中自己写
  3. EPS逻辑修改 考虑非管理组织的问题
  4. 登录成功之后页面不跳转的问题处理
  5. 附件逻辑修改 支持PC企业微信下载功能
  6. 门户模板添加 里面的逻辑需要项目上自己处理
  7. 扫描PC端二维码授权登录
  8. 向导支持简单多选
  9. 窗体卡片样式处理
  10. 修改密码已添加规则
  11. 表单checkbox样式问题处理
  12. 流程抄送人员实现

注意

此版本后端M3补丁请使用2021-2月份以后的版本, PC移动端的很多功能需要M3最新补丁， 请项目上尽量克服，尽快更新M3的最新补丁。以后不再考虑beta版和稳定版。对老的M3补丁不再继续做兼容。

2021-03-30-beta[版本]补丁包 提取码：l0l3

# 2021-01-30 beta版

  1. 登录机制修改，登录机制使用cookie的形式处理，主要后端控制
  2. 修改日期组件，扩大时间范围
  3. 扩展动态模块的数据类型，添加file类型，点击此数据，直接打开附件
  4. PHO模式，添加审批后事件，但事件需要在pho中自己编写
  5. 消息模块添加二维码功能
  6. 拍照获取base64图片功能bug修改
  7. 送审排他线处理
  8. 收藏模块，优化样式
  9. 添加表单自动返回功能，需要在config.js中添加autoCloseForm字段
  10. 监听APP从后台调到前台的功能，仅限安卓，IOS无法使用
  11. 优化数字类型的数据
  12. 修复流程中，老的委派模块checkbox不显示的问题

注意

此版本后端补丁请使用2020-12月份以后的版本

2021-01-30-beta[版本]补丁包 提取码：twv9
