# PowerMobile2 版本更新

注意

下面是更新版本的列表，更新版本的频率是1个月更新发布一次。

# 2020-05-30

  1. 解决新增保存后附件，拍照、图片、文件按钮不显示的bug
  2. 窗体filter字段问题 可对象 可数组
  3. 添加进度反馈标准功能。
  4. 送审模块在企业微信中的样式混乱问题。

# 2020-04-30

  1. pho页面的实现
  2. 基础组件全局封装
  3. 工作台菜单样式修改
  4. 添加计划反馈、计划查询
  5. 解决新增状态附件上传bug
  6. 同意模块选择人员时bug修复
  7. 流程中附件没有后缀问题修复
  8. 流程中打开附件会导致下载问题修复
2020-04-30[版本]补丁包 提取码：ct4t

# 2020-04-07

  1. 主表上传附件
  2. 字段上传附件
  3. 多语言
  4. 质量模块
  5. 文档模块
  6. 表单多选字段
2020-04-07[版本]补丁包 提取码：fcag

# 2020-03-17【紧急更新】

  1. 解决华为手机meta20、p20的兼容性问题
2020-03-17[版本]补丁包 提取码：bfa7

# 2020-02-30

  1. 工作台菜单显示列数自定义
  2. 调用原生外壳事件
  3. 窗体多条件查询
  4. 窗体禁止下面的新增 删除 操作按钮功能
  5. 表单禁止保存按钮功能
2020-02-30[版本]补丁包 提取码：fnk6

# 2020-01-30[版本]

  1. 优化通知模块，修改原来加载即已读的逻辑。手动触发已读，可单条，可全部。
  2. 修改审批流同意中的选择节点和人员的逻辑。
  3. 更新了付费版本的极光推送，注意要更新nodeServer 和 下载最新的APP(在应用市场下载)
2020-01-30[版本]补丁包 提取码：7wn3

# 2019-12-30[版本]

  1. 在表单中，嵌入iframe， 实现vue程序和iframe页面的方法进行交互。

2019-12-30[版本]补丁包

# 2019-11-30[版本]

  1. 通知模块 [界面优化]
  2. APP消息推送页面跳转优化 [跳转逻辑优化]
  3. 表单权限优化 [主表、子表、附件的编辑和禁止权限优化处理]
  4. 表单多字段input框 [添加双击事件, 弹出类似微信的蒙版]
  5. 审批流添加附件方法 [在审批流中可以上传附件]

2019-11-30[版本]补丁包

# 2019-10-30[版本]

  1. EPS数据的优化 [优化树形数据算法]
  2. 个人中心数据的迁移 [原来是后端控制，目前转义到前端进行控制]
  3. 添加允许头像上传的开关 [可以禁止上传头像]
  4. 消息模块的配置选择 [可以选择聊天模式和非聊天模式]
  5. 工作台菜单检索 [在工作台中可以进行搜索菜单名称]

2019-10-30[版本]补丁包

# 2019-09-30[初始版本]

  1. PowerMobile2的最初发布。
  2. 整个APP的实现
  3. 个人中心功能配置拓展
  4. 消息页面的集合添加
  5. 窗体和表单的实现
  6. 动态模块的开发
  7. 工作台模块的开发
  8. 企业微信的关联

2019-09-30[版本]补丁包
