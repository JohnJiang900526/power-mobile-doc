# 安装 Power-Mobile APP

# IOS系统

苹果手机，可以在APP Store中搜索下载PowerMobile2 即可，按照向导说明进行下载。

# 安卓系统

安卓手机，可以在华为、小米、vivo、OPPO、魅族的应用商店或者应用宝中，搜索PowerMobile2进行下载， 如果应用宝和华为商店中搜索到的APP名称叫PowerMobile, 但是安装成功后程序会自动改成PowerMobile2。
