# 安装 Power-Mobile APP

# IOS系统

苹果手机，可以在APP Store中搜索下载PowerMobile2 即可，按照向导说明进行下载。

# 安卓系统

安卓手机，可以在应用宝或者华为商店中，搜索PowerMobile进行下载，应用宝和华为 商店中搜索到的APP名称叫PowerMobile, 但是安装成功后程序会自动改成PowerMobile2。
