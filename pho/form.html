# 普通文本

  • 普通文本三种模式：可输入的文本，只读的文本，禁用的文本
  • 三种模式都很常见，根据自己的需要进行选择
// 可输入的文本
<div class="form-unit input-methods bottom-line">
  <div class="label">
    <p class="label-text">合同编号</p>
  </div>
  <div class="input-content">
    <div class="input-item">
      <input name="SubContractCode" type="text">
    </div>
  </div>
</div>

// 只读的文本
<div class="form-unit input-methods bottom-line">
  <div class="label">
    <p class="label-text">合同编号</p>
  </div>
  <div class="input-content mui-readonly">
    <div class="input-item">
      <input readonly name="SubContractCode" type="text">
    </div>
  </div>
</div>

// 禁用的文本
<div class="form-unit input-methods bottom-line">
  <div class="label">
    <p class="label-text">合同编号</p>
  </div>
  <div class="input-content mui-disabled">
    <div class="input-item">
      <input disabled name="SubContractCode" type="text">
    </div>
  </div>
</div>

# 多行文本框

  • 三种模式：可输入模式、只读模式、禁用模式
  • 一般常用可输入模式
// 可输入模式
<div class="form-unit textarea-methods">
  <div class="label">
    <p class="label-text">备注</p>
  </div>
  <textarea data-type="textarea" name="Meto" class="textarea">
    我是备注。
  </textarea>
</div>

// 只读模式
<div class="form-unit textarea-methods">
  <div class="label">
    <p class="label-text">备注</p>
  </div>
  <textarea data-type="textarea" readonly name="Meto" class="textarea" readonly>
    我是备注。
  </textarea>
</div>

// 禁用模式
<div class="form-unit textarea-methods">
  <div class="label">
    <p class="label-text">备注</p>
  </div>
  <textarea data-type="textarea" disabled name="Meto" class="textarea" disabled>
    我是备注。
  </textarea>
</div>

# checkbox 单选框文本

  • 三种模式：只读模式、禁用模式
  • 一般常用只读模式
// 只读模式
<div class="form-unit checkbox-methods bottom-line">
  <div class="mui-input-row mui-checkbox">
    <label>是否聚餐</label>
    <input data-type="checkbox" name="isParty" value="Item" type="checkbox" checked>
  </div>
</div>

// 禁用模式
<div class="form-unit checkbox-methods bottom-line">
  <div class="mui-input-row mui-checkbox">
    <label>是否聚餐</label>
    <input data-type="checkbox" disabled name="isParty" value="Item" type="checkbox">
  </div>
</div>

# 日期文本

  • 有两种模式：只读模式(可以出发click事件)，禁用模式(不出发click事件)
  • 一般是只读模式
// 非禁用日期
<div class="form-unit input-methods bottom-line">
  <div class="label">
    <p class="label-text">分摊日期</p>
  </div>
  <div class="input-content mui-readonly">
    <div class="input-item">
      <input data-type="datePicker" readonly name="AllocatedDate" type="text">
    </div>
    <div class="input-item after-icon">
      <span class="icon mui-icon-extra mui-icon-extra-calendar"></span>
    </div>
  </div>
</div>

// 禁用日期
<div class="form-unit input-methods">
  <div class="label">
    <p class="label-text">录入日期</p>
  </div>
  <div class="input-content mui-disabled">
    <div class="input-item">
      <input data-type="datePicker" disabled name="RegDate" type="text">
    </div>
    <div class="input-item after-icon">
      <span class="icon mui-icon-extra mui-icon-extra-calendar"></span>
    </div>
  </div>
</div>

# 下拉框文本

  • 有两种模式：只读模式(可以出发click事件)，禁用模式(不出发click事件)
  • 一般是只读模式，但是表单状态的这种字段一般都是禁用模式
// 只读模式
<div class="form-unit input-methods bottom-line">
  <div class="label">
    <p class="label-text">合同类别</p>
  </div>
  <div class="input-content mui-readonly">
    <div class="input-item">
      <input data-type="select" readonly name="SubContractType" type="text">
    </div>
    <div class="input-item after-icon">
      <span class="icon mui-icon mui-icon-arrowdown"></span>
    </div>
  </div>
</div>

// 禁用模式
<div class="form-unit input-methods bottom-line">
  <div class="label">
    <p class="label-text">审批状态</p>
  </div>
  <div class="input-content mui-disabled">
    <div class="input-item">
      <input data-type="select" disabled name="Status" type="text">
    </div>
    <div class="input-item after-icon">
      <span class="icon mui-icon mui-icon-arrowdown"></span>
    </div>
  </div>
</div>
